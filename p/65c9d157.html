GitHub 配置秘钥

完成上面的教程会报错，我们现在来配置密钥

我们把私钥放到我们存放 Hexo 原始文件的代码仓库里面，用于触发 Actions 时使用。

把公钥放到 GitHub Pages 对应的代码仓库里面，用于 Hexo 部署时的写入工作。

1.配置私钥

首先在 GitHub 上打开保存Hexo源码的仓库，访问 Settings -> Secrets and variables -> Actions，点击New repository secret，name填HEXO_DEPLOY_KEY,Secret填写github-deploy-key文件里的内容也就是填入私钥

2.添加公钥

接下来我们需要访问存放网页的仓库，也就是 Hexo 部署以后的仓库，比如：yourname.github.io 这种，访问 Settings -> Deploy keys,按 Add deploy key 来添加一个新的公钥，在 Title 中输入HEXO_DEPLOY_PUB ，在 Key 中粘贴 github-deploy-key.pub 文件的内容。
